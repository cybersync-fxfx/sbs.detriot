# Workspace Index

Indexed on 2026-04-23.

## Overview

This repository is now a broader Detroit SBS workspace rather than just a panel app.

Current major pieces:

- a root Node.js + Express + WebSocket control server
- a React + Vite frontend served from `frontend/dist`
- a dynamic per-user installer generated by the backend
- a standalone versioned agent bundle in `agent/`
- guard-host bootstrap and hardening scripts
- a threat-radar background scanner plus Supabase schema for it

The repo is still centered on the SBS control plane, but it now also contains more direct operator tooling and agent lifecycle scripts.

## Runtime Shape

1. A browser hits the panel domain and loads the React frontend from the Express server.
2. Unauthenticated users land on a public landing page and then move to login.
3. Authenticated users use the dashboard over same-origin `/api/*` requests plus a WebSocket connection.
4. Remote agents register, send telemetry, poll for commands, and post command results.
5. The backend caches last known stats per user, broadcasts events over WebSocket, and runs a periodic threat-radar scan on the guard host.
6. Operator shell scripts can inspect connected agents, manage blocklists, harden the guard, or update the bundled agent.
