Installation Steps

Follow these steps to deploy the SBS agent on your Ubuntu server. You must have root access.

Step 1: Download the installer script

Step 2: Upload to your server

scp sbs-agent-XXXX.sh root@YOUR_SERVER_IP:/root/
Step 3: Run the installer

sudo bash /root/sbs-agent-XXXX.sh
